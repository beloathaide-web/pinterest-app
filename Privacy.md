# Política de Privacidade

**Última atualização:** 08 de julho de 2026

Esta Política de Privacidade descreve como este aplicativo ("o aplicativo") coleta, usa e protege informações ao se conectar com sua conta do Pinterest.

## 1. Sobre o aplicativo

Este é um aplicativo de uso pessoal, desenvolvido e operado por Athaide Belo da Silva Santos, com a finalidade de automatizar a publicação de Pins na própria conta do Pinterest do desenvolvedor, referentes a produtos em promoção disponíveis em sua vitrine de afiliado.

O aplicativo não é destinado a uso por terceiros e não oferece cadastro, login ou qualquer funcionalidade para outros usuários além do próprio desenvolvedor.

## 2. Quais dados são acessados

Para funcionar, o aplicativo solicita acesso à API do Pinterest com os seguintes escopos:

- **boards:read / boards:write** — para listar e criar conteúdo nos boards da própria conta
- **pins:read / pins:write** — para criar Pins na própria conta

O aplicativo acessa apenas dados da conta do Pinterest do próprio desenvolvedor (nome dos boards, IDs de boards e Pins), necessários exclusivamente para a criação automatizada de publicações.

O aplicativo **não coleta, armazena ou processa dados pessoais de terceiros**, não acessa informações de outros usuários do Pinterest, e não compartilha dados com nenhum serviço externo além da própria API do Pinterest e da API oficial de afiliados da Shopee (usada apenas para obter links de produtos e informações de preço/desconto).

## 3. Como os dados são usados

Os dados de autenticação (token de acesso e token de atualização) são usados exclusivamente para:

- Autenticar chamadas à API do Pinterest em nome da própria conta do desenvolvedor
- Publicar Pins contendo título, descrição, imagem e link de produto

Nenhum dado é vendido, alugado ou compartilhado com terceiros para fins de marketing ou publicidade.

## 4. Armazenamento e segurança

Os tokens de acesso são armazenados de forma privada (variáveis de ambiente ou gerenciador de segredos), acessíveis apenas pelo desenvolvedor, e nunca são expostos publicamente ou incluídos em código-fonte compartilhado.

## 5. Divulgação de afiliado

Os Pins publicados pelo aplicativo contêm links de afiliado da Shopee. Isso significa que o desenvolvedor pode receber uma comissão por compras realizadas através desses links, sem custo adicional para o comprador. Essa informação é divulgada diretamente na descrição de cada Pin publicado.

## 6. Retenção e exclusão de dados

Os tokens de acesso podem ser revogados a qualquer momento diretamente nas configurações da conta do Pinterest (Aplicativos autorizados). Ao revogar o acesso, o aplicativo perde permissão para realizar qualquer ação na conta.

## 7. Alterações nesta política

Esta política pode ser atualizada periodicamente. A data da última atualização está indicada no topo desta página.
